## 偏好与习惯

### 沟通风格
极简主义。提问通常简短，省略主语和背景，要求 AI 具备极强的上下文理解能力，能够结合历史对话补全意图。

### 代码审美
厌恶"屎山"代码。倾向于彻底重构而非打补丁。

### 文档需求
重度依赖文档自动化。常需将网站内容、代码逻辑转化为 .doc 或 .xlsx 格式。

### 安全意识
对文件操作（特别是删除）极其敏感，必须经过确认。
