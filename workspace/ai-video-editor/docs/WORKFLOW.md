## 2. 静音检测详细流程

```
输入视频
    │
    ▼
┌─────────────────────┐
│  提取音频轨道        │
│  (FFmpeg -vn -ac 1) │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  计算音频样本能量    │
│  - 分帧处理          │
│  - 计算 RMS 能量     │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  应用阈值过滤        │
│  - 低于阈值 = 静音   │
│  - 高于阈值 = 有声   │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  合并相邻静音段      │
│  - 小间隔合并         │
│  - 应用边缘保留       │
└─────────────────────┘
    │
    ▼
输出：静音片段列表
```

### 静音检测算法说明

1. **音频分帧**：将音频分成每帧 20ms 的片段
2. **能量计算**：计算每帧的 RMS 能量值
3. **阈值判断**：能量 < threshold * max_energy → 静音
4. **边缘保留**：在静音片段前后保留 margin 秒
5. **最小长度**：静音片段需大于最小时长才被标记

## 4. 字幕生成详细流程

```
输入视频
    │
    ▼
┌─────────────────────┐
│  提取音频轨道        │
│  (FFmpeg -vn -ac 1) │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  语音识别 (Whisper)  │
│  - 加载模型          │
│  - 转录音频          │
│  - 生成时间轴        │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  生成 SRT 字幕文件   │
│  - 格式化时间码      │
│  - 添加序号          │
│  - 文本分行          │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│  嵌入字幕到视频      │
│  (FFmpeg -vf subs)   │
└─────────────────────┘
    │
    ▼
输出：带字幕视频
```

### SRT 字幕格式示例

```
1
00:00:01,000 --> 00:00:03,000
大家好，欢迎观看本期视频

2
00:00:03,500 --> 00:00:05,500
今天我们来聊聊 AI 视频剪辑

3
00:00:06,000 --> 00:00:08,000
让我们开始吧
```

## 5. 错误处理流程

```
操作执行
    │
    ├──────────────────────────┐
    │                          │
    ▼                          ▼
┌─────────────┐        ┌─────────────┐
│   成功      │        │   失败      │
└─────────────┘        └─────────────┘
                              │
                              ▼
                    ┌─────────────────┐
                    │  捕获异常类型    │
                    └─────────────────┘
                              │
          ┌───────────────────┼───────────────────┐
          │                   │                   │
          ▼                   ▼                   ▼
    ┌───────────┐       ┌───────────┐       ┌───────────┐
    │ 可恢复错误 │       │ 用户错误   │       │ 系统错误   │
    │ (重试)    │       │ (提示用户) │       │ (记录日志) │
    └───────────┘       └───────────┘       └───────────┘
          │                   │                   │
          ▼                   ▼                   ▼
    ┌───────────┐       ┌───────────┐       ┌───────────┐
    │ 等待重试   │       │ 返回错误码 │       │ 清理资源   │
    │ 最多3次   │       │ 和说明    │       │ 终止程序   │
    └───────────┘       └───────────┘       └───────────┘
```

### 错误分类与处理

| 错误类型 | 示例 | 处理方式 |
|----------|------|----------|
| 可恢复 | 网络超时、临时文件占用 | 等待后重试，最多3次 |
| 用户错误 | 文件不存在、格式不支持 | 返回错误码，提示用户 |
| 系统错误 | 内存溢出、磁盘满 | 记录日志，清理资源，终止 |
