## 开发规范

### 参数配置原则
**问题：** 之前参数硬编码在代码里，不方便修改

**原则：**
1. **所有可调参数必须放配置文件**，不能硬编码在代码里
2. 配置文件用 YAML 格式，清晰易读
3. 代码中读取配置，并提供默认值兜底
4. 用户修改配置文件即可调整参数，无需改代码

**示例：**
```yaml
# config.yaml
video_type_detection:
  thresholds:
    interview:
      face_count_min: 2
      speech_ratio_min: 0.7
```

```python
# 代码中读取
thresholds = self.config.get("video_type_detection", {}).get("thresholds", {})
t = thresholds.get("interview", {})
face_min = t.get("face_count_min", 2)  # 有默认值兜底
```

**后续开发必须遵循此原则！**
