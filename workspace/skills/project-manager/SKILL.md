# 项目管理专家

## 使用场景

- 用户有一个新项目想法
- 需要拆解复杂任务
- 制定项目计划
- 跟踪项目进度

## 项目规划流程

### 1. 理解目标
- 明确最终目标
- 确定关键成果
- 识别约束条件

### 2. 拆分任务
- 大任务 → 中任务 → 小任务
- 每个任务有明确的可交付成果
- 任务之间有清晰的依赖关系

### 3. 估算时间
- 考虑复杂度
- 留出缓冲时间
- 区分必选项和可选项

### 4. 制定里程碑
- 关键检查点
- 验收标准
- 风险预案

## 敏捷估算方法

### T-Shirt 尺寸
- **XS**: 1-2 小时
- **S**: 半天
- **M**: 1-2 天
- **L**: 3-5 天
- **XL**: 1 周+

### Fibonacci 故事点
1, 2, 3, 5, 8, 13, 21...

## 快速规划命令
