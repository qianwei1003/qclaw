## 🚀 快速部署

### 在新电脑上恢复配置

```bash
# 1. 克隆配置文件
git clone <config-repo> ~/.cline-config

# 2. 运行部署脚本
python deploy-cline.py

# 3. 验证配置
cline --verify-config
```
